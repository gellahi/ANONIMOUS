Data Persistence Multi-Page App

Explore different web storage APIs through these three applications

Multi-Step Registration Form

A two-step registration form that uses localStorage to persist data between steps.

  • Step 1: Enter name and email
  • Step 2: Create password and confirm
  • Data persists even if you close the browser
Try Registration Form

Shopping Cart System

A shopping cart implementation that uses cookies to store selected items.

  • Browse and add products to your cart
  • View your cart with selected items
  • Data persists with configurable expiration
Try Shopping Cart

Quiz Application

A simple quiz app that uses sessionStorage to track progress and score.

  • Answer multiple-choice questions
  • See your final score
  • Data is cleared when browser is closed
Try Quiz App
